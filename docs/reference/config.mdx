# Configuration

<Note>
  The plugin reads configuration from <code>channels.openclaw-simplex</code>. Managed mode is gone; use <code>connection.wsUrl</code> for the separately running runtime.
</Note>

## Top-level channel config

```json
{
  "channels": {
    "openclaw-simplex": {
      "enabled": true,
      "dmPolicy": "pairing",
      "allowFrom": ["*"],
      "groupPolicy": "allowlist",
      "groupAllowFrom": [],
      "connection": {
        "wsUrl": "ws://127.0.0.1:5225",
        "connectTimeoutMs": 15000
      },
      "accounts": {}
    }
  }
}
```

## Useful fields

| Field | Meaning |
| --- | --- |
| `enabled` | Master toggle |
| `dmPolicy` | `open` \| `allowlist` \| `pairing` \| `disabled` |
| `allowFrom` | DM allowlist entries |
| `groupPolicy` | `open` \| `allowlist` \| `disabled` |
| `groupAllowFrom` | Group allowlist entries |
| `mediaMaxMb` | Maximum accepted/uploaded media size in MB |
| `connection.wsUrl` | WebSocket URL for the running `simplex-chat` WebSocket API |
| `connection.wsHost` / `connection.wsPort` | Optional host/port form for local endpoints |
| `accounts` | Per-account overrides |
