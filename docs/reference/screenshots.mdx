# Screenshots

This page shows the supported SimpleX pairing and approval flow.

Generate or retrieve the invite/address link in `simplex-chat` with `/c`, `/ad`, or `/show_address`, then continue with the pairing flow below.

If you need automation instead, OpenClaw also exposes `simplex.invite.create` and `simplex.invite.list`.

It assumes the SimpleX channel is already configured with a reachable `channels.openclaw-simplex.connection.wsUrl`.

## Pairing flow

1. Send the first message from the SimpleX app and capture the pairing request.

<img src="/images/pairing-request.png" alt="Pairing code shown in SimpleX app" width="720" />

2. Review the pairing request in OpenClaw.

<img src="/images/pairing-list.png" alt="OpenClaw pairing requests table" width="900" />

3. Approve pairing in OpenClaw.

```bash
openclaw pairing list
openclaw pairing approve openclaw-simplex <code>
```

4. Confirm the approval and test the first successful chat.
