# Runtime Connection

OpenClaw no longer starts `simplex-chat` itself. Run the SimpleX runtime separately and connect over WebSocket.

## Recommended shape

```json
{
  "channels": {
    "openclaw-simplex": {
      "connection": {
        "wsUrl": "ws://127.0.0.1:5225"
      }
    }
  }
}
```

## Notes

- Install `simplex-chat` on the host.
- Start the SimpleX runtime yourself.
- Point `channels.openclaw-simplex.connection.wsUrl` at the running endpoint.
- Invite/address creation can still be done directly in the `simplex-chat` console with `/c`, `/ad`, `/show_address`, and `/delete_address`.
