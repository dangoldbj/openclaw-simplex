# Setup

Prerequisite: install `simplex-chat` first (see [Getting Started](/guide/getting-started)).

## Runtime

Run `simplex-chat` separately and point OpenClaw at its WebSocket endpoint.

Start the runtime:

```bash
simplex-chat -p 5225
```

```json
{
  "channels": {
    "openclaw-simplex": {
      "connection": {
        "wsUrl": "ws://127.0.0.1:5225"
      }
    }
  }
}
```

If `simplex-chat` is not running, OpenClaw will keep the channel disconnected and surface the connection error in channel status.

Manual invite and address management still happens in the `simplex-chat` console:

```text
/c
/ad
/show_address
/delete_address
```

## Per-account config

```json
{
  "channels": {
    "openclaw-simplex": {
      "accounts": {
        "ops": {
          "enabled": true,
          "connection": {
            "wsUrl": "ws://127.0.0.1:7777"
          }
        }
      }
    }
  }
}
```
