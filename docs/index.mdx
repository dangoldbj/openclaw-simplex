# OpenClaw SimpleX

SimpleX channel plugin for OpenClaw.

<Warning>
  Breaking in <code>1.0.0</code>: the plugin id and channel id are now <code>openclaw-simplex</code>, managed mode was removed, and existing users should run <code>openclaw simplex migrate</code>. See <a href="/guide/migration">Migration</a>.
</Warning>

This plugin brings OpenClaw to SimpleX with clear policy controls, multi-account support, and an external WebSocket runtime model.

## Quick Start

Fresh install:

```bash
curl -o- https://raw.githubusercontent.com/simplex-chat/simplex-chat/stable/install.sh | bash
simplex-chat -p 5225
openclaw plugins install @dangoldbj/openclaw-simplex
openclaw plugins enable openclaw-simplex
openclaw channels add --channel openclaw-simplex --url ws://127.0.0.1:5225
```

For the full setup flow, trust configuration, pairing, invite generation, and persistent runtime setup, see [Getting Started](/guide/getting-started) and [Setup](/guide/setup).

Upgrading from older `simplex` ids?

```bash
openclaw simplex migrate
```

See [Migration](/guide/migration) for the full breaking-change checklist.
