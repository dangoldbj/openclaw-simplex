# OpenClaw SimpleX

SimpleX channel plugin for OpenClaw.

> Breaking in `1.0.0`: the plugin id and channel id are now `openclaw-simplex`, managed mode was removed, and existing users should run `openclaw simplex migrate`. See [Migration](/guide/migration).

This plugin brings OpenClaw to SimpleX with clear policy controls, multi-account support, and an external WebSocket runtime model.

## Quick Start

Fresh install:

1. Install `simplex-chat`.
2. Start the WebSocket runtime:

```bash
simplex-chat -p 5225
```

3. Install and enable the plugin:

```bash
openclaw plugins install @dangoldbj/openclaw-simplex
openclaw plugins enable openclaw-simplex
```

4. Configure the channel:

```bash
openclaw channels add --channel openclaw-simplex --url ws://127.0.0.1:5225
```

For the full setup flow, trust configuration, pairing, and invite generation, see [Getting Started](/guide/getting-started).

Upgrading from the older `simplex` ids?

```bash
openclaw simplex migrate
```

See [Migration](/guide/migration) for the full breaking-change checklist.
